# PrivateMessagesGUI
Shows the sent messages of the [Private Messages](https://umod.org/plugins/private-messages) plugin as game tips.

## Required Plugin
[Private Messages](https://umod.org/plugins/private-messages)

## Configuration
```json
{
  "Game Tip Duration In Seconds": 10.0,
  "Show To Sender": false,
  "Show To Receiver": true
}
```
